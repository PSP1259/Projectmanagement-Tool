# Java Project Management Tool

This is a Java Swing-based Project Management application, developed as part of the Software Engineering course at Kalaidos University Zürich.

## Features
* **Task Management:** Create, edit, and track tasks (Open/Done).
* **Time Tracking:** Track time spent on tasks with automatic visual warnings (> 60 minutes).
* **Team Collaboration:** Assign tasks to team members (generates dynamic UI badges).
* **Communication:** Add timestamped comments to tasks.
* **Filtering & Sorting:** Filter tasks dynamically by status and assignees.
* **Persistent Storage:** Data is saved using Java Properties/XML.
* **FAQ/Help:** Integrated FAQ and help section for a brief overview and user guidance.

## Architecture
The application is built following the **Model-View-Controller (MVC)** architectural pattern to ensure an object-oriented design.
